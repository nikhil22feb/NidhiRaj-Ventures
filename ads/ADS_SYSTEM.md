
# Ads System (Scalable)

## Channels
- Meta (Instagram + Facebook)
- Google Shopping (later)

## Creative Framework
Hook (0-2s) -> Problem -> Solution -> Offer -> CTA

## Budgeting
Start ₹500/day per product; scale winners to ₹3000+/day

## Metrics
- CTR > 2%
- CPC ₹2–₹6
- CPA target = margin/2

## Scaling
Duplicate ad sets with new creatives; keep audience broad
